Field guide

Running the session (~20 min intro)

Start with AND, OR, NOT and fan-out — one signal can branch and feed several gates. Build one nontrivial function together (XOR works well). Tell them, without proving it yet, that these three gates are enough to build any Boolean function.

The main event

There are 16 Boolean functions of two variables. Everyone hunts in the Workbench; solved ones go on the Board. Most groups land around 13/16 in an hour. The constants (all-0 and all-1) are usually last — that's normal, they're supposed to be annoying.

If they're flying through it

  • Three variables. Head to tab 04 · Extra Time — six specific targets, not all 256.
  • Completeness. Ask why AND/OR/NOT can build any truth table. Someone will usually stumble onto the DNF idea on their own.
  • De Morgan, minimality, NAND-only. Good stretch questions if you have time left.

Using this in a classroom

The Board lives in your browser's storage — no server, no accounts. Easiest setup: one machine on the projector runs the Board; students build on their own laptops and either walk up to submit or call out their circuit. Set your name once at the top and every submission carries it.
